# Bash — компактная шпаргалка по горячим клавишам

| Раздел | Клавиши | Действие |
|--------|---------|----------|
| Перемещение | Ctrl+A | Начало строки |
| | Ctrl+E | Конец строки |
| | Alt+B | Назад слово |
| | Alt+F | Вперёд слово |
| | Ctrl+B | Назад символ |
| | Ctrl+F | Вперёд символ |
| Команды | Ctrl+L | Очистить экран |
| | Ctrl+S | Стоп вывода |
| | Ctrl+Q | Продолжить вывод |
| | Ctrl+C | Прервать команду |
| | Ctrl+Z | Пауза / в фон (`fg`) |
| Редактирование | Ctrl+U | Удалить до начала |
| | Ctrl+K | Удалить до конца |
| | Ctrl+W | Удалить слово слева |
| | Alt+D | Удалить слово справа |
| | Ctrl+D | Удалить символ |
| | Ctrl+H | Удалить слева (backspace) |
| | Ctrl+Y | Вставить вырезанное |
| Манипуляции | Alt+C | Слово — Заглавная |
| | Alt+U | Слово — ВЕРХНИЙ регистр |
| | Alt+L | Слово — нижний регистр |
| | Alt+T | Поменять слова |
| | Ctrl+T | Поменять символы |
| История | Ctrl+R | Поиск по истории |
| | Ctrl+G | Выход из поиска |
| | Ctrl+P | Пред. команда |
| | Ctrl+N | След. команда |
| | Alt+. | Последнее слово |
| Bang (!) | !! | Последняя команда |
| | !foo | Команда, нач. с foo |
| | !foo:p | Показать, не выполнять |
| | !$ | Последний аргумент |
| | !* | Все аргументы |
| | !^ | Первый аргумент |

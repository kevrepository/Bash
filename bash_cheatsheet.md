# Bash — компактная шпаргалка по горячим клавишам

| Раздел                | Сочетание      | Описание                                   |
|-----------------------|----------------|--------------------------------------------|
| **Перемещение**       | `Ctrl + A`     | В начало строки                            |
|                       | `Ctrl + E`     | В конец строки                             |
|                       | `Alt + B`      | К началу предыдущего слова                 |
|                       | `Alt + F`      | К концу следующего слова                   |
|                       | `Ctrl + B`     | Назад на символ                            |
|                       | `Ctrl + F`     | Вперёд на символ                           |
| **Управление командой** | `Ctrl + L`   | Очистить экран                             |
|                       | `Ctrl + S`     | Остановить вывод                           |
|                       | `Ctrl + Q`     | Возобновить вывод                          |
|                       | `Ctrl + C`     | Прервать команду                           |
|                       | `Ctrl + Z`     | Пауза/в фон (`fg` — вернуть)              |
| **Вырезать/вставить** | `Ctrl + U`     | Удалить до начала строки                   |
|                       | `Ctrl + K`     | Удалить до конца строки                    |
|                       | `Ctrl + W`     | Удалить слово слева                        |
|                       | `Alt + D`      | Удалить слово справа                       |
|                       | `Ctrl + D`     | Удалить символ под курсором                |
|                       | `Ctrl + H`     | Удалить символ слева                       |
|                       | `Ctrl + Y`     | Вставить последнее вырезанное              |
| **Манипуляция текстом** | `Alt + C`   | Слово от курсора с Заглавной               |
|                       | `Alt + U`      | Слово от курсора В ВЕРХНЕМ РЕГИСТРЕ        |
|                       | `Alt + L`      | Слово от курсора в нижнем регистре         |
|                       | `Alt + T`      | Поменять местами текущее и предыдущее слово|
|                       | `Ctrl + T`     | Поменять местами символы                   |
| **История**           | `Ctrl + R`     | Поиск по истории                           |
|                       | `Ctrl + G`     | Выход из поиска                            |
|                       | `Ctrl + P`     | Предыдущая команда                         |
|                       | `Ctrl + N`     | Следующая команда                          |
|                       | `Alt + .`      | Последнее слово прошлой команды            |
| **Bang-команды**      | `!!`           | Повторить последнюю команду                |
|                       | `!foo`         | Последняя команда, начинающаяся с `foo`    |
|                       | `!foo:p`       | Показать, но не выполнять                  |
|                       | `!$`           | Последнее слово предыдущей команды         |
|                       | `!*`           | Все аргументы предыдущей команды           |
|                       | `!^`           | Первый аргумент предыдущей команды         |
